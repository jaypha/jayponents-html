# Jayponents HTML

Written by Jason den Dulk

Use Jayponents to create HTML elements.

### Features

- Create HTML elements using templates without having to provide to boilerplate.


## Requirements

PHP v5.4.0 or greater.
Jaypha/Jayponents

## Installation

```
composer require jaypha/jayponents
```
